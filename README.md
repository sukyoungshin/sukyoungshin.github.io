# Portfolio app

React를 이용하여 만든 개인 포트폴리오 앱 입니다.

## PROJECT 소개

🔗 URL : [LIVE DEMO](https://sukyoungshin.github.io/), [깃허브코드](https://github.com/sukyoungshin/githubpage)

### Features
- Contact: 폼 제출 시, 연동된 google sheet에 입력된 데이터를 POST전송 (API활용) <br/>
